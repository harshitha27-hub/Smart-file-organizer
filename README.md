Smart File Organizer

A Python-based smart file organizer that automatically organizes files,
detects duplicate files, monitors folders, and provides analytics reports.

## Features
- Automatically organizes files into folders
- Detects and removes duplicate files
- Monitors folders for new files
- Generates analytics and reports
- Simple dashboard support

## Technologies Used
- Python
- OS module
- Shutil
- Watchdog
- SQLite (if applicable)

## How to Run the Project
1. Clone the repository:
   git clone https://github.com/harshitha27-hub/Smart-file-organizer

2. Install required packages:
   pip install -r requirements.txt

3. Run the project:
   python main.py

## Project Structure
- file_organizer.py – organizes files
- duplicate_detector.py – detects duplicate files
- folder_monitor.py – monitors folders
- analytics.py – generates analytics
- dashboard.py – displays dashboard
